# ПОЛНЫЙ РАЗБОР КОДА ДЛЯ НОВИЧКОВ

Этот код — простая программа для создания и просмотра почтовых аккаунтов Gmail.

---

## 1. **Заголовки и библиотеки**
```cpp
#include <iostream>
#include <vector>

using namespace std;
```

- `#include <iostream>`: Подключает библиотеку для работы с консолью (ввод/вывод данных).
- `#include <vector>`: Подключает библиотеку для работы с динамическими массивами (векторами).
- `using namespace std;`: Позволяет не писать `std::` перед стандартными функциями (например, `std::cout` становится просто `cout`).

**Почему важно**: Эти строки нужны для того, чтобы программа могла работать с данными и выводить их на экран.

---

## 2. **Класс `Gmail`**
```cpp
class Gmail {
    string name;
    string gmailn;

public:
    Gmail(string userName, string userEmail) {
        name = userName;
        gmailn = userEmail;
    }

    void displayInfo() const {
        cout << "Имя пользователя: " << name << endl;
        cout << "Почта: " << gmailn << endl;
    }

    string getName() const {
        return name;
    }

    string getEmail() const {
        return gmailn;
    }
};
```

### Что это?
Класс `Gmail` описывает одного пользователя Gmail. У него есть два поля:
- `name` — имя пользователя.
- `gmailn` — почтовый адрес.

### Методы класса:
1. **Конструктор**:
   ```cpp
   Gmail(string userName, string userEmail)
   ```
   - Когда создаётся новый объект класса, этот метод автоматически заполняет поля `name` и `gmailn`.
   - Например, если мы пишем `Gmail("Alice", "alice@gmail.com")`, то `name` станет `"Alice"`, а `gmailn` — `"alice@gmail.com"`.

2. **Метод `displayInfo()`**:
   - Выводит информацию о пользователе в консоль.
   - Например:
     ```
     Имя пользователя: Alice
     Почта: alice@gmail.com
     ```

3. **Геттеры (`getName` и `getEmail`)**:
   - Эти методы позволяют получить доступ к закрытым полям (`name` и `gmailn`) извне класса.
   - Например, если у нас есть объект `user`, то `user.getName()` вернёт имя пользователя.

**Почему важно**: Класс помогает организовать данные и действия с ними в одном месте. Это делает программу более структурированной.

## 3. **Основная программа (`main`)**

### 3.1. Объявление переменных
```cpp
vector<Gmail> users;
setlocale(0, "RU");
```

- `vector<Gmail> users;`:
  - Создаёт пустой вектор (динамический массив), который будет хранить объекты класса `Gmail`.
  - По мере добавления новых пользователей они будут попадать в этот вектор.

- `setlocale(0, "RU");`:
  - Настройка локали для корректного отображения русских символов в консоли.

**Почему важно**: Вектор позволяет хранить сколько угодно пользователей, а локаль делает вывод красивым.

---

### 3.2. Главное меню
```cpp
while (true) {
    cout << endl << "----------------------" << endl;
    cout << "   Gmail - Почта" << endl;
    cout << "----------------------" << endl;
    cout << "[1] Создать почту" << endl;
    cout << "[2] Написать сообщение" << endl;
    cout << "[3] Посмотреть сохранённые почты" << endl;
    cout << "[4] Сохранить данные" << endl;
    cout << "[0] Выйти из программы" << endl;
    cout << "----------------------" << endl;
    cout << "Выберите опцию: ";
    char opt;
    cin >> opt;
```

- Программа работает в бесконечном цикле (`while (true)`), пока пользователь не выберет выход (`opt == '0'`).
- Выводится меню с опциями, и программа ждёт, что пользователь введёт номер действия.

**Почему важно**: Меню делает программу удобной для использования.

---

### 3.3. Выход из программы
```cpp
if (opt == '0') {
    system("cls");
    cout << "----------------------" << endl;
    cout << "| Выход из программы |" << endl;
    cout << "----------------------" << endl << endl << endl;
    break;
}
```

- Если пользователь вводит `'0'`, программа очищает экран (`system("cls")`) и завершается (`break`).

---

### 3.4. Создание нового аккаунта
```cpp
else if (opt == '1') {
    string name;
    string gmailname;
    system("cls");
    cout << "-------------------------" << endl;
    cout << "     Создание почты" << endl;
    cout << "-------------------------" << endl;
    cout << "Введите имя пользователя: ";
    cin >> name;
    cout << endl << "Введите имя почты (без @gmail.com): ";
    cin >> gmailname;
    users.push_back(Gmail(name, gmailname + "@gmail.com"));
    system("cls");
    cout << "--------------------------" << endl;
    cout << "| Почта успешно создана! |" << endl;
    cout << "--------------------------" << endl;
    cout << "Имя пользователя: " << name << endl;
    cout << "Почта " << name << ": " << gmailname << "@gmail.com" << endl;
    cout << "--------------------------" << endl;
}
```

- Программа спрашивает имя пользователя и название почты (без `@gmail.com`).
- Создаётся новый объект класса `Gmail` с помощью конструктора:
  ```cpp
  Gmail(name, gmailname + "@gmail.com")
  ```
- Этот объект добавляется в вектор `users` с помощью `push_back`.

**Почему важно**: Это основная функция программы — добавление новых пользователей.

---

### 3.5. Просмотр всех аккаунтов
```cpp
else if (opt == '3') {
    system("cls");
    cout << "-------------------------" << endl;
    cout << "     Список почт" << endl;
    cout << "-------------------------" << endl;

    if (users.empty()) {
        cout << "Нет сохранённых почтовых аккаунтов." << endl;
    }
    else {
        for (const auto& user : users) {
            user.displayInfo();
            cout << "-------------------------" << endl;
        }
    }
}
```

- Если вектор `users` пуст (`users.empty()`), программа сообщает об этом.
- Если вектор не пуст, программа проходит по всем элементам с помощью цикла:
  ```cpp
  for (const auto& user : users)
  ```
  - `const auto& user`: Эффективный способ обратиться к каждому элементу вектора без копирования.
  - Для каждого пользователя вызывается метод `displayInfo()`, который выводит его данные.

**Почему важно**: Цикл позволяет быстро вывести все аккаунты, а проверка на пустоту предотвращает ошибки.

---

### 3.6. Обработка неверных опций
```cpp
else {
    system("cls");
    cout << "----------------------------------------" << endl;
    cout << "| Неверная опция, попытайтесь ещё раз! |" << endl;
    cout << "----------------------------------------" << endl << endl;
}
```

- Если пользователь вводит что-то, кроме `0`, `1`, `2`, `3` или `4`, программа сообщает об ошибке.


## ОСНОВНЫЕ ПОНЯТИЯ, КОТОРЫЕ НУЖНО ЗАПОМНИТЬ

1. **Классы**:
   - Класс `Gmail` хранит данные о пользователе и предоставляет методы для работы с этими данными.

   - Вектор `users` хранит всех созданных пользователей.

   - Цикл `for (const auto& user : users)` проходит по всем элементам вектора эффективно.

4. **Меню**:
   - Программа использует меню для взаимодействия с пользователем.
